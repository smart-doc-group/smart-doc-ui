<h1 align="center">Smart-Doc-UI Project</h1>

## Introduce
smart-doc-ui是smart-doc开源API文档生成工具官方开发的文档渲染UI。smart-doc-ui类似于swagger ui,
但是比swagger ui更佳漂亮和易用。

## Features
- 简单易用、漂亮简洁。
- 支持渲染open api文档。
